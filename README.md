# Modul Pembelajaran Matematik Hermes

## Ringkasan
Modul ini merangkumi geometri koordinat yang komprehensif termasuk sistem koordinat, formula jarak, konsep titik tengah, dan aplikasi penyelesaian masalah praktikal.

## Objektif Pembelajaran
- Memahami sistem koordinat Cartesian
- Mengira jarak antara dua titik
- Mencari titik tengah segmen garis
- Menyelesaikan masalah geometri koordinat dunia sebenar
- Mengaplikasikan konsep kepada bentuk geometri dan rajah

## Struktur Modul
1. **Pengenalan Koordinat** - Konsep asas dan penentuan kedudukan
2. **Jarak Antara Titik** - Formula dan pengiraan
3. **Konsep Titik Tengah** - Mencari titik tengah dan aplikasi
4. **Penyelesaian Masalah** - Aplikasi lanjutan dan masalah geometri

## Prasyarat
- Pemahaman asas garisan nombor
- Kepuasan dengan nombor positif dan negatif
- Kemahiran algebra asas

## Pendekatan Pengajaran
Setiap bahagian termasuk:
- Penerangan yang jelas dengan rajah
- Contoh langkah demi langkah
- Soalan latihan untuk pelajar
- Aplikasi dunia sebenar
- Bahan penilaian

## Pengagihan Masa
- Jumlah modul: 4-6 jam
- Setiap bahagian: 1-1.5 jam
- Amalan dan penilaian: 1-2 jam
