# Hermes Math Learning Modules

## Overview
This repository contains comprehensive mathematics learning materials designed for tuition classes. All learning materials are in Malay language to support local education needs.

## Current Modules
- **Coordinate Geometry** - Complete module covering coordinate systems, distance formulas, midpoint concepts, and real-world applications

## Repository Structure
```
topics/
├── coordinate-geometry/
│   ├── README.md              # Module overview
│   ├── lesson-notes/          # Comprehensive lessons
│   ├── worksheets/            # Student practice materials
│   ├── assessments/           # Evaluation tests
│   └── diagrams/             # Visual aids
└── [future modules]
```

## Features
- **Comprehensive Coverage**: Each topic includes detailed lesson notes, examples, practice problems, and assessments
- **Step-by-Step Solutions**: All problems include complete worked solutions
- **Real-World Applications**: Practical examples connecting mathematical concepts to everyday situations
- **Bilingual Support**: Learning materials in Malay, project documentation in English
- **Structured Learning**: Progressive difficulty from basic to advanced concepts

## Usage
1. Navigate to `topics/[module-name]/` for specific topic materials
2. Use `lesson-notes/` for teaching preparation
3. Assign `worksheets/` for student practice
4. Use `assessments/` for evaluation
5. Refer to `diagrams/` for visual learning aids
