# Classroom Setup Guide - Hermes Math Learning Modules

## Overview
This guide provides step-by-step instructions for setting up and using the Hermes Math Learning Modules in a classroom environment.

## Prerequisites
- Computer with internet access
- Web browser (Chrome, Firefox, Safari, Edge)
- Projector or smartboard for display
- Student devices (optional, for interactive learning)
- PDF reader for printing materials

## Quick Setup (5 minutes)

### 1. Repository Access
```bash
# Clone the repository
git clone https://github.com/hafizn24/hermes-math.git
cd hermes-math
```

### 2. File Organization
```
hermes-math/
├── topics/
│   └── coordinate-geometry/
│       ├── README.md              # Module overview
│       ├── lesson-notes/          # Teacher preparation
│       ├── worksheets/            # Student practice
│       ├── assessments/           # Tests and evaluations
│       └── diagrams/             # Visual aids
├── README.md                     # Main project documentation
└── classroom-setup.md           # This file
```

### 3. Material Preparation
- Print worksheets from `worksheets/` folder
- Review lesson notes from `lesson-notes/` folder
- Prepare assessment materials from `assessments/` folder
- Download diagrams from `diagrams/` folder

## Detailed Lesson Plans

### Lesson 1: Introduction to Coordinate System (60 minutes)

#### Materials Needed
- Coordinate system diagrams
- Student worksheets (MS122A)
- Whiteboard and markers

#### Lesson Flow
1. **Introduction (10 minutes)**
   - Explain the concept of coordinate system
   - Show interactive diagrams
   - Demonstrate plotting points

2. **Interactive Demo (15 minutes)**
   - Use projector to display coordinate system
   - Demonstrate plotting sample points
   - Show quadrant identification

3. **Student Practice (25 minutes)**
   - Distribute MS122A worksheet
   - Students plot points on coordinate plane
   - Teacher circulates to assist

4. **Review (10 minutes)**
   - Discuss common mistakes
   - Reinforce key concepts
   - Preview next lesson

#### Teaching Tips
- Use real-world examples (maps, GPS, games)
- Emphasize the relationship between coordinates and position
- Use visual aids extensively

### Lesson 2: Distance Between Points (60 minutes)

#### Materials Needed
- Distance formula diagrams
- Practice problems worksheet
- Calculator for complex calculations

#### Lesson Flow
1. **Review (10 minutes)**
   - Quick quiz on coordinate system
   - Review previous lesson concepts

2. **Concept Introduction (15 minutes)**
   - Introduce distance formula
   - Show derivation using Pythagorean theorem
   - Demonstrate with examples

3. **Interactive Demo (15 minutes)**
   - Use interactive distance formula diagram
   - Show step-by-step calculations
   - Demonstrate special cases (same x or y coordinates)

4. **Student Practice (15 minutes)**
   - Students work on practice problems
   - Teacher provides guidance
   - Students check answers

5. **Application (5 minutes)**
   - Show real-world applications
   - Discuss practical uses

### Lesson 3: Midpoint Concept (60 minutes)

#### Materials Needed
- Midpoint diagrams
- Real-world application examples
- Student worksheets

#### Lesson Flow
1. **Introduction (10 minutes)**
   - Introduce midpoint concept
   - Show real-world examples (bridges, paths)
   - Demonstrate visual representation

2. **Formula Derivation (15 minutes)**
   - Show midpoint formula
   - Explain averaging concept
   - Demonstrate with examples

3. **Interactive Demo (15 minutes)**
   - Use interactive midpoint diagram
   - Show midpoint calculation
   - Demonstrate finding unknown endpoints

4. **Student Practice (15 minutes)**
   - Students work on midpoint problems
   - Teacher provides guidance
   - Students apply to real-world scenarios

5. **Summary (5 minutes)**
   - Review key concepts
   - Preview next lesson

### Lesson 4: Problem Solving Applications (90 minutes)

#### Materials Needed
- Advanced practice problems
- Real-world scenario cards
- Assessment materials

#### Lesson Flow
1. **Review (15 minutes)**
   - Quick review of previous concepts
   - Address any remaining questions

2. **Advanced Concepts (30 minutes)**
   - Complex perimeter calculations
   - Area calculations using coordinates
   - Circle equations and properties

3. **Real-world Applications (30 minutes)**
   - Drone delivery routes
   - GPS navigation
   - Urban planning examples

4. **Group Activity (15 minutes)**
   - Students work in groups on scenarios
   - Present solutions to class

## Differentiation Strategies

### For Advanced Students
- Provide additional complex problems
- Introduce 3D coordinate systems
- Explore advanced geometric concepts

### For struggling students
- Provide visual aids and step-by-step guides
- Use manipulatives and hands-on activities
- Break down complex problems into smaller steps

### For English Language Learners
- Provide bilingual materials
- Use visual extensively
- Demonstrate concepts with real examples

## Common Challenges and Solutions

### Challenge 1: Students struggle with negative coordinates
**Solution**: Use color-coding and physical demonstrations

### Challenge 2: Distance formula confusion
**Solution**: Use Pythagorean theorem visualization and step-by-step breakdown

### Challenge 3: Midpoint concept abstraction
**Solution**: Use real-world examples and interactive demonstrations
