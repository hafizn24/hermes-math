# 7.1 Pengenalan Sistem Koordinat

## Objektif Pembelajaran
- Memahami sistem koordinat Cartesian
- Mengenal pasti kedudukan menggunakan koordinat (x, y)
- Plot titik pada satah koordinat
- Menentukan jarak mengufuk dan mencancang

## Sistem Koordinat Cartesian

Sistem koordinat Cartesian terdiri daripada:
- **Paksi-x** - garisan mengufok
- **Paksi-y** - garisan mencancang
- **Asalan (0, 0)** - titik di mana paksi bersilang
- **Empat sukuan** - bernombor I, II, III, IV

### Notasi Koordinat
Satu titik ditulis sebagai **(x, y)** di mana:
- **x** = jarak mengufok dari asalan
- **y** = jarak mencancang dari asalan

### Sistem Sukuan
- **Sukuan I**: (positif, positif)
- **Sukuan II**: (negatif, positif)  
- **Sukuan III**: (negatif, negatif)
- **Sukuan IV**: (positif, negatif)

## Contoh

### Contoh 1: Plot Titik
**Masalah**: Plot titik berikut pada satah koordinat:
- A(2, 3)
- B(-1, 4)
- C(3, -2)
- D(-3, -1)

**Penyelesaian**:
1. Mula dari asalan (0, 0)
2. Untuk titik A(2, 3):
   - Bergerak 2 unit ke kanan pada paksi-x
   - Bergerak 3 unit ke atas pada paksi-y
   - Tandakan titik A
3. Untuk titik B(-1, 4):
   - Bergerak 1 unit ke kiri pada paksi-x
   - Bergerak 4 unit ke atas pada paksi-y
   - Tandakan titik B
4. Teruskan untuk semua titik

### Contoh 2: Mengenal Pasti Sukuan
**Masalah**: Kenal pasti sukuan untuk setiap titik:
- P(5, 2)
- Q(-3, 6)
- R(-4, -1)
- S(7, -3)

**Penyelesaian**:
- P(5, 2): Sukuan I (kedua-dua positif)
- Q(-3, 6): Sukuan II (x negatif, y positif)
- R(-4, -1): Sukuan III (kedua-dua negatif)
- S(7, -3): Sukuan IV (x positif, y negatif)

## Soalan Amalan

### Soalan 1: Plot Titik
**Tugasan**: Plot titik berikut pada satah koordinat:
- M(1, 5)
- N(-2, 3)
- O(4, -1)
- P(-3, -4)

**Penyelesaian Bertutor**:
1. Lukis paksi koordinat dengan asalan
2. Untuk M(1, 5): 1 unit kanan, 5 unit atas
3. Untuk N(-2, 3): 2 unit kiri, 3 unit atas
4. Untuk O(4, -1): 4 unit kanan, 1 unit bawah
5. Untuk P(-3, -4): 3 unit kiri, 4 unit bawah

### Soalan 2: Mencari Koordinat
**Tugasan**: Tentukan koordinat titik A, B, C, D pada satah koordinat: A ialah 3 unit ke kanan dan 2 unit ke atas dari asalan; B ialah 2 unit ke kiri dan 4 unit ke atas dari asalan; C ialah 4 unit ke kanan dan 3 unit ke bawah dari asalan; D ialah 4 unit ke kiri dan 1 unit ke bawah dari asalan. Tentukan koordinat A, B, C, D.

**Penyelesaian Bertutor**:
- Titik A: 3 unit kanan, 2 unit atas → A(3, 2)
- Titik B: 2 unit kiri, 4 unit atas → B(-2, 4)
- Titik C: 4 unit kanan, 3 unit bawah → C(4, -3)
- Titik D: 4 unit kiri, 1 unit bawah → D(-4, -1)

### Soalan 3: Aplikasi Dunia Sebenar
**Tugasan**: Seorang perancang bandar merancang taman. Pintu masuk di (0, 0). Air pancut di (3, 2), playground di (-1, 4), dan tandasan di (2, -1). Plot lokasi ini.

**Penyelesaian Bertutor**:
1. Lukis satah koordinat mewakili taman
2. Plot pintu masuk di asalan
3. Plot air pancut: 3 unit timur, 2 unit utara
4. Plot playground: 1 unit barat, 4 unit utara
5. Plot tandasan: 2 unit timur, 1 unit selatan
