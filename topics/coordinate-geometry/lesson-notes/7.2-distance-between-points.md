# 7.2 Jarak Antara Dua Titik

## Objektif Pembelajaran
- Mengira jarak antara dua titik menggunakan formula jarak
- Mengaplikasikan teorem Pythagoras kepada geometri koordinat
- Mencari jarak apabila titik berkongsi x atau y yang sama
- Menyelesaikan masalah berkaitan jarak

## Formula Jarak

Jarak antara dua titik \(A(x_1, y_1)\) dan \(B(x_2, y_2)\) ialah:

\[ AB = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \]

### Kes Khas
1. **Koordinat-x sama**: Jarak = \(|y_2 - y_1|\)
2. **Koordinat-y sama**: Jarak = \(|x_2 - x_1|\)

## Contoh

### Contoh 1: Pengiraan Jarak Asas
**Masalah**: Cari jarak antara A(2, 3) dan B(5, 7)

**Penyelesaian**:
1. Kenal pasti koordinat:
   - \(A(x_1, y_1) = (2, 3)\)
   - \(B(x_2, y_2) = (5, 7)\)
2. Gunakan formula jarak:
   \[ AB = \sqrt{(5 - 2)^2 + (7 - 3)^2} \]
   \[ AB = \sqrt{3^2 + 4^2} \]
   \[ AB = \sqrt{9 + 16} \]
   \[ AB = \sqrt{25} \]
   \[ AB = 5 \text{ unit} \]

### Contoh 2: Koordinat-x Sama
**Masalah**: Cari jarak antara P(3, 1) dan Q(3, 6)

**Penyelesaian**:
1. Kedua-dua titik mempunyai koordinat-x yang sama (3)
2. Jarak = \(|y_2 - y_1|\)
3. Jarak = \(|6 - 1| = 5\) unit

### Contoh 3: Koordinat-y Sama
**Masalah**: Cari jarak antara R(-2, 4) dan S(7, 4)

**Penyelesaian**:
1. Kedua-dua titik mempunyai koordinat-y yang sama (4)
2. Jarak = \(|x_2 - x_1|\)
3. Jarak = \(|7 - (-2)| = |9| = 9\) unit

### Contoh 4: Koordinat Negatif
**Masalah**: Cari jarak antara A(-1, -2) dan B(3, 4)

**Penyelesaian**:
1. Kenal pasti koordinat:
   - \(A(x_1, y_1) = (-1, -2)\)
   - \(B(x_2, y_2) = (3, 4)\)
2. Gunakan formula jarak:
   \[ AB = \sqrt{(3 - (-1))^2 + (4 - (-2))^2} \]
   \[ AB = \sqrt{(4)^2 + (6)^2} \]
   \[ AB = \sqrt{16 + 36} \]
   \[ AB = \sqrt{52} \]
   \[ AB = 2\sqrt{13} \approx 7.21 \text{ unit} \]

## Soalan Amalan

### Soalan 1: Jarak Asas
**Tugasan**: Cari jarak antara titik P(1, 2) dan Q(4, 6)

**Penyelesaian Bertutor**:
1. Kenal pasti: \(P(x_1, y_1) = (1, 2)\), \(Q(x_2, y_2) = (4, 6)\)
2. Gunakan formula: \(PQ = \sqrt{(4 - 1)^2 + (6 - 2)^2}\)
3. Kira: \(PQ = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\) unit

### Soalan 2: Koordinat-x Sama
**Tugasan**: Cari jarak antara A(-3, 1) dan B(-3, 5)

**Penyelesaian Bertutor**:
1. Kedua-dua mempunyai x = -3
2. Jarak = \(|5 - 1| = 4\) unit

### Soalan 3: Koordinat-y Sama
**Tugasan**: Cari jarak antara C(2, -1) dan D(8, -1)

**Penyelesaian Bertutor**:
1. Kedua-dua mempunyai y = -1
2. Jarak = \(|8 - 2| = 6\) unit

### Soalan 4: Pengiraan Perimeter
**Tugasan**: Segi tiga ABC mempunyai bucu A(1, 1), B(3, 4), C(5, 1). Cari perimeternya.

**Penyelesaian Bertutor**:
1. Cari AB:
   \[ AB = \sqrt{(3-1)^2 + (4-1)^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.61 \]
2. Cari BC:
   \[ BC = \sqrt{(5-3)^2 + (1-4)^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.61 \]
3. Cari AC:
   \[ AC = \sqrt{(5-1)^2 + (1-1)^2} = \sqrt{16 + 0} = 4 \]
4. Perimeter = AB + BC + AC = 3.61 + 3.61 + 4 = 11.22 unit

### Soalan 5: Mencari Koordinat Tidak Diketahui
**Tugasan**: Titik A(2, 3) dan titik B(x, 7) berjarak 5 unit. Cari x.

**Penyelesaian Bertutor**:
1. Gunakan formula jarak:
   \[ 5 = \sqrt{(x - 2)^2 + (7 - 3)^2} \]
   \[ 5 = \sqrt{(x - 2)^2 + 16} \]
2. Kuasa dua kedua-dua belah:
   \[ 25 = (x - 2)^2 + 16 \]
3. Selesaikan untuk x:
   \[ (x - 2)^2 = 9 \]
   \[ x - 2 = \pm 3 \]
   \[ x = 5 \text{ atau } x = -1 \]
