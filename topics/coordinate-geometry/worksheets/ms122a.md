# Lembaran Kerja MS122A - Kedudukan Bandar

## Tujuan
Mengenal pasti kedudukan suatu titik menggunakan koordinat

## Bahan: Lembaran kerja (grid koordinat disertakan di bawah)

### Arahan:
1. Gunakan grid koordinat di bawah (1 kotak = 1 unit).
2. Bandar-bandar berikut telah ditetapkan pada koordinat berikut untuk tujuan latihan ini (bukan koordinat geografi sebenar):
   - Batu Pahat: (2, 1)
   - Kluang: (6, 4)
   - Segamat: (10, 8)
3. Dengan menggabungkan jarak mengufuk dan mencancang, sahkan kedudukan setiap bandar di atas grid.

## Soalan Amalan

### Soalan 1: Plot Kedudukan Bandar
**Tugasan**: Plot koordinat berikut pada grid:
- Batu Pahat: (2, 1)
- Kluang: (6, 4)
- Segamat: (10, 8)

### Soalan 2: Kedudukan Relatif
**Tugasan**: Bandar X terletak 5 unit ke timur dan 3 unit ke utara dari asalan. Cari koordinat bandar X.

**Penyelesaian Bertutor**:
1. Jarak mengufuk (x) = 5 unit ke timur
2. Jarak mencancang (y) = 3 unit ke utara
3. Koordinat: (5, 3)

### Soalan 3: Jarak Antarabandar
**Tugasan**: Kira jarak antara:
(a) Batu Pahat dan Kluang
(b) Kluang dan Segamat

**Penyelesaian Bertutor**:
(a) Jarak Batu Pahat(2,1) ke Kluang(6,4):
\[ \text{Jarak} = \sqrt{(6-2)^2 + (4-1)^2} = \sqrt{16+9} = \sqrt{25} = 5 \text{ unit} \]

(b) Jarak Kluang(6,4) ke Segamat(10,8):
\[ \text{Jarak} = \sqrt{(10-6)^2 + (8-4)^2} = \sqrt{16+16} = \sqrt{32} = 4\sqrt{2} \approx 5.66 \text{ unit} \]

### Soalan 4: Peta Koordinat
**Tugasan**: Lukis peta koordinat dan plot kedudukan ketiga-tiga bandar tersebut. Sambungkan mereka dengan garis lurus.

### Soalan 5: Aplikasi Dunia Sebenar
**Tugasan**: Sebuah syarikat pengangkutan perlu menghantar barangan dari Batu Pahat ke Kluang, kemudian ke Segamat. Cari jumlah jarak perjalanan.

**Penyelesaian Bertutor**:
1. Jarak Batu Pahat ke Kluang = 5 unit (dari Soalan 3a)
2. Jarak Kluang ke Segamat = \(4\sqrt{2} \approx 5.66\) unit (dari Soalan 3b)
3. Jumlah jarak = 5 + 5.66 = 10.66 unit

### Soalan 6: Titik Tengah Perjalanan
**Tugasan**: Cari titik tengah bagi perjalanan dari Batu Pahat ke Segamat. Di mana letak titik ini?

**Penyelesaian Bertutor**:
1. Batu Pahat(2,1), Segamat(10,8)
2. Gunakan formula titik tengah:
\[ M\left( \frac{2+10}{2}, \frac{1+8}{2} \right) = M(6, 4.5) \]
3. Titik ini mewakili lokasi tengah geometri perjalanan (nota: ini adalah titik tengah matematik, bukan semestinya lokasi rehat praktikal sebenar).

## Tip untuk Guru
- Galakkan pelajar menggunakan grid kertas untuk plot titik
- Nota: koordinat bandar dalam lembaran kerja ini adalah untuk tujuan latihan sahaja, bukan koordinat geografi sebenar
- Gunakan skala yang munasabah untuk koordinat
- Tekankan konsep jarak mengufuk dan mencancang
- Sambungkan dengan aplikasi navigasi GPS sebagai perbincangan lanjutan
