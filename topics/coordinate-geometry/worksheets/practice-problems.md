## Jawapan

### Bahagian A
**Soalan 1**: Plot titik pada satah koordinat
- A(2, 3): 2 unit kanan, 3 unit atas
- B(-1, 4): 1 unit kiri, 4 unit atas
- C(3, -2): 3 unit kanan, 2 unit bawah
- D(-3, -1): 3 unit kiri, 1 unit bawah

**Soalan 2**: Sukuan
- P(5, 2): Sukuan I
- Q(-3, 6): Sukuan II
- R(-4, -1): Sukuan III
- S(7, -3): Sukuan IV

**Soalan 3**: Jarak
(a) PQ = \(\sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = 5\) unit
(b) AB = \(|5 - 1| = 4\) unit (koordinat-x sama)
(c) CD = \(|8 - 2| = 6\) unit (koordinat-y sama)
(d) EF = \(\sqrt{(3-(-1))^2 + (4-(-2))^2} = \sqrt{16 + 36} = \sqrt{52} \approx 7.21\) unit

**Soalan 4**: Titik tengah
(a) \(\left( \frac{1+5}{2}, \frac{2+8}{2} \right) = (3, 5)\)
(b) \(\left( \frac{-2+4}{2}, \frac{-3+1}{2} \right) = (1, -1)\)
(c) \(\left( \frac{3+7}{2}, \frac{4+6}{2} \right) = (5, 5)\)

### Bahagian B
**Soalan 5**: \(5 = \sqrt{(x-2)^2 + (7-3)^2}\)
\(25 = (x-2)^2 + 16\)
\((x-2)^2 = 9\)
\(x = 5\) atau \(x = -1\)

**Soalan 6**: 
(a) Perimeter = \(3.61 + 3.61 + 4 = 11.22\) unit
(b) Segi tiga sama kaki
(c) Pusat graviti: \((3, 2)\)

**Soalan 7**:
(a) \(AC = \sqrt{(5-1)^2 + (6-2)^2} = \sqrt{32} = 4\sqrt{2}\)
(b) Titik tengah: \((3, 4)\)
(c) Luas = 16 unit persegi

**Soalan 8**:
(a) Jejari = 5 unit
(b) \((x-3)^2 + (y-4)^2 = 25\)
(c) Q(-1, 7)

### Bahagian C
**Soalan 9**: Pusat graviti: \((4, 3.33)\)

**Soalan 10**:
(a) Perimeter = \(2\sqrt{32} + 8\) unit
(b) Titik tengah AB: \((0, 1)\), Titik tengah BC: \((4, 1)\)
(c) D(2, 7)

**Soalan 11**: Jumlah jarak = 15.65 unit

**Soalan 12**: 
- Pintu masuk ke air pancut: \((1.5, 1)\)
- Playground ke kedai: \((1, 1.5)\)
- Taman permainan ke taman rehat: \((3.5, 1)\)

### Bahagian D

**Soalan 14**:
(a) PQ = 5, QR = 5, PR = 6
(b) Segi tiga sama kaki
(c) Titik tengah PQ: \((2.5, 3)\), QR: \((5.5, 3)\), PR: \((4, 1)\)

**Soalan 15**: 
- Jarak ke paksi-x: 4 unit
- Jarak ke paksi-y: 3 unit

**Soalan 16**: B(6, 8), Jarak AB = \(\sqrt{(6-2)^2 + (8-4)^2} = \sqrt{32} = 4\sqrt{2}\)
