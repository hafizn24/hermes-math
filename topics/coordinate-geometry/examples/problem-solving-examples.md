# Contoh Penyelesaian Masalah

## Contoh 1: Sifat Segi Tiga
**Masalah**: Segi tiga ABC mempunyai bucu A(1, 1), B(3, 4), C(5, 1). Cari:
(a) Perimeter segi tiga ABC
(b) Jenis segi tiga
(c) Koordinat pusat graviti

**Penyelesaian Langkah demi Langkah**:

**(a) Perimeter**:
1. Cari AB:
   \[ AB = \sqrt{(3-1)^2 + (4-1)^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.61 \]
2. Cari BC:
   \[ BC = \sqrt{(5-3)^2 + (1-4)^2} = \sqrt{4 + 9} = \sqrt{13} \approx 3.61 \]
3. Cari AC:
   \[ AC = \sqrt{(5-1)^2 + (1-1)^2} = \sqrt{16 + 0} = 4 \]
4. Perimeter = AB + BC + AC = 3.61 + 3.61 + 4 = 11.22 unit

**(b) Jenis segi tiga**:
- AB = BC = 3.61, jadi ia adalah segi tiga sama kaki
- Semak jika segi tiga bersudut tegak: AB² + BC² = 13 + 13 = 26, AC² = 16
- 26 ≠ 16, jadi bukan segi tiga bersudut tegak
- **Jawapan**: Segi tiga sama kaki

**(c) Pusat graviti**:
1. Formula pusat graviti: \(G\left( \frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3} \right)\)
2. \(G\left( \frac{1+3+5}{3}, \frac{1+4+1}{3} \right)\)
3. \(G(3, 2)\)

**Jawapan**: 
- Perimeter: 11.22 unit
- Jenis: Segi tiga sama kaki
- Pusat graviti: (3, 2)

## Contoh 2: Sifat Segi Empat
**Masalah**: Segi empat tepat ABCD mempunyai bucu A(2, 3), B(6, 3), C(6, 7), D(2, 7). Cari:
(a) Panjang pepenjuru AC
(b) Titik tengah pepenjuru AC
(c) Luas segi empat tepat

**Penyelesaian Langkah demi Langkah**:

**(a) Pepenjuru AC**:
1. A(2, 3), C(6, 7)
2. \(AC = \sqrt{(6-2)^2 + (7-3)^2} = \sqrt{16 + 16} = \sqrt{32} = 4\sqrt{2} \approx 5.66\)

**(b) Titik tengah AC**:
1. \(M\left( \frac{2+6}{2}, \frac{3+7}{2} \right)\)
2. \(M(4, 5)\)

**(c) Luas**:
1. Panjang AB = |6-2| = 4
2. Lebar AD = |7-3| = 4
3. Luas = 4 × 4 = 16 unit persegi

**Jawapan**:
- Pepenjuru: \(4\sqrt{2}\) unit
- Titik tengah: (4, 5)
- Luas: 16 unit persegi

## Contoh 3: Sifat Bulatan
**Masalah**: Bulatan mempunyai pusat C(3, 4) dan melalui titik P(7, 1). Cari:
(a) Jejari bulatan
(b) Persamaan bulatan
(c) Titik Q yang bertentangan dengan P

**Penyelesaian Langkah demi Langkah**:

**(a) Jejari**:
1. Jarak CP = jejari
2. \(CP = \sqrt{(7-3)^2 + (1-4)^2} = \sqrt{16 + 9} = \sqrt{25} = 5\)

**(b) Persamaan bulatan**:
1. \((x - h)^2 + (y - k)^2 = r^2\)
2. \((x - 3)^2 + (y - 4)^2 = 25\)

**(c) Titik Q**:
1. Q bertentangan dengan P, jadi C ialah titik tengah PQ
2. Letak Q(x, y)
3. \(3 = \frac{7 + x}{2}\) → \(6 = 7 + x\) → \(x = -1\)
4. \(4 = \frac{1 + y}{2}\) → \(8 = 1 + y\) → \(y = 7\)
5. Q(-1, 7)

**Jawapan**:
- Jejari: 5 unit
- Persamaan: \((x - 3)^2 + (y - 4)^2 = 25\)
- Titik Q: (-1, 7)

## Contoh 4: Aplikasi Dunia Sebenar
**Masalah**: Drone perlu menghantar paket ke tiga lokasi: A(1, 2), B(4, 6), C(7, 2). Cari titik permulaan optimum untuk mengurangkan jumlah jarak.

**Penyelesaian Langkah demi Langkah**:

1. **Cari pusat graviti** (titik optimum untuk jarak sama):
   \[ G\left( \frac{1+4+7}{3}, \frac{2+6+2}{3} \right) \]
   \[ G(4, 3.33) \]

2. **Kira jarak dari G ke setiap lokasi**:
   - GA = \(\sqrt{(4-1)^2 + (3.33-2)^2} = \sqrt{9 + 1.77} = \sqrt{10.77} \approx 3.28\)
   - GB = \(\sqrt{(4-4)^2 + (3.33-6)^2} = \sqrt{0 + 7.13} = \sqrt{7.13} \approx 2.67\)
   - GC = \(\sqrt{(4-7)^2 + (3.33-2)^2} = \sqrt{9 + 1.77} = \sqrt{10.77} \approx 3.28\)

3. **Jumlah jarak**:
   Jumlah jarak = 3.28 + 2.67 + 3.28 = 9.23 unit

**Jawapan**: Titik permulaan optimum ialah (4, 3.33) dengan jumlah jarak 9.23 unit.

## Contoh 5: Geometri Kompleks
**Masalah**: Titik A(-2, 3), B(2, -1), C(6, 3). Cari:
(a) Perimeter segi tiga ABC
(b) Titik tengah AB dan BC
(c) Titik D untuk membuat ABCD segi empar

**Penyelesaian Langkah demi Langkah**:

**(a) Perimeter**:
1. AB = \(\sqrt{(2-(-2))^2 + (-1-3)^2} = \sqrt{16 + 16} = \sqrt{32}\)
2. BC = \(\sqrt{(6-2)^2 + (3-(-1))^2} = \sqrt{16 + 16} = \sqrt{32}\)
3. AC = \(\sqrt{(6-(-2))^2 + (3-3)^2} = 8\)
4. Perimeter = \(2\sqrt{32} + 8\)

**(b) Titik tengah**:
1. Titik tengah AB: \(\left( \frac{-2+2}{2}, \frac{3+(-1)}{2} \right) = (0, 1)\)
2. Titik tengah BC: \(\left( \frac{2+6}{2}, \frac{-1+3}{2} \right) = (4, 1)\)

**(c) Titik D**:
1. Dalam segi empar, titik tengah AC = titik tengah BD
2. Titik tengah AC: \(\left( \frac{-2+6}{2}, \frac{3+3}{2} \right) = (2, 3)\)
3. Letak D(x, y): \((2, 3) = \left( \frac{2+x}{2}, \frac{-1+y}{2} \right)\)
4. Selesaikan:
   - \(2 = \frac{2+x}{2}\) → \(4 = 2+x\) → \(x = 2\)
   - \(3 = \frac{-1+y}{2}\) → \(6 = -1+y\) → \(y = 7\)
5. D(2, 7)

**Jawapan**:
- Perimeter: \(2\sqrt{32} + 8\) unit
- Titik tengah AB: (0, 1), Titik tengah BC: (4, 1)
- Titik D: (2, 7)

## Soalan Amalan untuk Pelajar

### Soalan 1: Analisis Segi Tiga
**Tugasan**: Segi tiga PQR mempunyai bucu P(1, 1), Q(4, 5), R(7, 1). Cari:
(a) Perimeter
(b) Jenis segi tiga
(c) Pusat graviti

**Penyelesaian Bertutor**:
(a) Perimeter:
1. PQ = \(\sqrt{(4-1)^2 + (5-1)^2} = \sqrt{9 + 16} = 5\)
2. QR = \(\sqrt{(7-4)^2 + (1-5)^2} = \sqrt{9 + 16} = 5\)
3. PR = \(\sqrt{(7-1)^2 + (1-1)^2} = 6\)
4. Perimeter = 5 + 5 + 6 = 16 unit

(b) Jenis: Segi tiga sama kaki (PQ = QR)

(c) Pusat graviti: \(\left( \frac{1+4+7}{3}, \frac{1+5+1}{3} \right) = (4, 2.33)\)

### Soalan 2: Sifat Segi Empat
**Tugasan**: Segi empat tepat ABCD mempunyai A(1, 2), B(5, 2), C(5, 6), D(1, 6). Cari:
(a) Panjang pepenjuru
(b) Titik tengah pepenjuru
(c) Luas

**Penyelesaian Bertutor**:
(a) Pepenjuru AC: \(\sqrt{(5-1)^2 + (6-2)^2} = \sqrt{16 + 16} = \sqrt{32} = 4\sqrt{2}\)
(b) Titik tengah: \((3, 4)\)
(c) Luas: 4 × 4 = 16 unit persegi

### Soalan 3: Bulatan dan Garis
**Tugasan**: Bulatan pusat (2, 3) dengan jejari 5. Cari titik di mana bulatan bersilang dengan paksi-x.

**Penyelesaian Bertutor**:
1. Pada paksi-x, y = 0
2. Persamaan: \((x-2)^2 + (0-3)^2 = 25\)
3. \((x-2)^2 + 9 = 25\)
4. \((x-2)^2 = 16\)
5. x-2 = ±4
6. x = 6 atau x = -2
7. Titik: (6, 0) dan (-2, 0)

### Soalan 4: Laluan Optimum
**Tugasan**: Penghantar perlu melawat kedai di A(1, 1), B(3, 4), C(6, 2). Cari titik permulaan optimum.

**Penyelesaian Bertutor**:
1. Pusat graviti: \(\left( \frac{1+3+6}{3}, \frac{1+4+2}{3} \right) = (3.33, 2.33)\)
2. Kira jarak dari pusat graviti ke setiap kedai
3. Jumlah jarak untuk jumlah penghantaran

### Soalan 5: Geometri Kompleks
**Tugasan**: Titik A(-2, 3), B(2, -1), C(6, 3). Cari:
(a) Perimeter segi tiga
(b) Titik tengah AB dan BC
(c) Titik D untuk membuat ABCD segi empar

**Penyelesaian Bertutor**:
(a) Perimeter:
1. AB = \(\sqrt{(2-(-2))^2 + (-1-3)^2} = \sqrt{16 + 16} = \sqrt{32}\)
2. BC = \(\sqrt{(6-2)^2 + (3-(-1))^2} = \sqrt{16 + 16} = \sqrt{32}\)
3. AC = \(\sqrt{(6-(-2))^2 + (3-3)^2} = 8\)
4. Perimeter = \(2\sqrt{32} + 8\)

(b) Titik tengah:
1. Titik tengah AB: \((0, 1)\)
2. Titik tengah BC: \((4, 1)\)

(c) Titik D:
1. Dalam segi empar, titik tengah AC = titik tengah BD
2. Titik tengah AC: \((2, 3)\)
3. Letak D(x, y): \((2, 3) = \left( \frac{2+x}{2}, \frac{-1+y}{2} \right)\)
4. Selesaikan: x = 2, y = 7
5. D(2, 7)
